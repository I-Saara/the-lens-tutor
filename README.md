🔭 The Lens: Adaptive Agentic Tutor
Don't just learn. See the world through your own lens.
The Lens is an advanced AI-driven educational platform built for the Google Cloud: Build with AI Agentic Premier League (Challenge 2). It solves the "one-size-fits-all" learning crisis by dynamically re-architecting complex STEM concepts into cinematic metaphors based on a user’s specific passions.

🚀 Why "The Lens" Wins
Most AI tutors are static chatbots. The Lens is a sophisticated Multi-Agent Orchestration Engine that maps unfamiliar technical data onto a user's existing mental models to reduce cognitive load and increase retention.

✨ Key Features
🎭 Multi-Persona Switching: Instantly toggle the entire curriculum between Trekker, MasterChef, Founder, and Pro-Gamer modes.

🎁 The "Surprise" Engine: The ultimate proof of agency. Enter any niche hobby (e.g., K-Pop, F1 Racing, Cricket) and the agent performs zero-shot pedagogical mapping in real-time.

🎬 Cinematic AI Lessons (Veo 3.1): Integrated with Google Veo 3.1 to generate cinematic video analogies that visualize abstract concepts (e.g., explaining Latency through a Pit-Stop animation).

🔍 Translation Toggle: A unique UI feature allowing users to hover over metaphorical terms to reveal the underlying technical documentation and "ground" their learning.

🛡️ Agentic Guardrails: A background "Fact-Checker" agent audits the Teacher's narrative to ensure 100% technical accuracy.

🛠️ Technical Architecture
The system utilizes a 4-stage Agentic Workflow built on the Google Cloud stack:

The Orchestrator (Gemini 1.5 Flash): Analyzes user intent and generates a structured semantic mapping JSON.

The Teacher (Gemini 1.5 Pro): Crafts a personalized narrative lesson using the mapping.

The Visualizer (Veo 3.1 for video generation / Imagen 3 for image generation): Generates high-fidelity video and image assets to anchor the metaphor visually.

The Fact-Checker (Gemini 1.5 Flash): Audits the output against technical documentation to prevent hallucinations.
